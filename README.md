# KNN Classification Project

This repository contains a Jupyter Notebook implementing the **K-Nearest Neighbors (KNN)** algorithm for classification tasks, created as part of the RINEX Data Science training.

## Project Overview
- Implements KNN from scratch and/or using `sklearn`.
- Includes data preprocessing, model training, and evaluation.
- Visualizes results for better understanding.

## Requirements
Install dependencies with:
```bash
pip install -r requirements.txt
```

## Files
- `RINEX KNN.ipynb`: Jupyter Notebook containing the implementation.
- `README.md`: Project documentation (this file).

## Usage
Run the notebook in Jupyter:
```bash
jupyter notebook 'RINEX KNN.ipynb'
```
